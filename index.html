Taskcluster Tools for Firefox Translations

The following are a list of Taskcluster tools for Firefox translations, but they could be useful for other Taskcluster projects.

Benchmark Performance

Run a translations performance benchmark

Cost Estimator

Estimate the cost of training runs

Artifacts Size

Get the size of artifacts

Graphs

Get some live graphs of training

Released Models

View the currently released Firefox translations models

Marian Model Inspector

View the architecture of Marian Models

Training Dashboard

Track multiple training runs

Task Profiler

Visualize a task's logs with the Firefox Profiler

Visualizer

Visualize a training graph using a force directed graph
